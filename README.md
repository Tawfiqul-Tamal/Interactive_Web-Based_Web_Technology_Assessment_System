# Interactive Web-Based Web Technology Assessment System

An interactive quiz application developed using **HTML5, CSS3, and JavaScript** to assess users' knowledge of Web Technology concepts. The system provides instant feedback, automatic scoring, and a user-friendly interface for an engaging learning experience.

---

## 📌 Project Overview

The Interactive Web-Based Web Technology Assessment System is a browser-based quiz platform designed to evaluate fundamental web technology knowledge through multiple-choice questions. The application dynamically generates quiz questions, validates answers, calculates scores, and displays performance results instantly.

This project demonstrates practical implementation of front-end web development concepts, including DOM manipulation, event handling, responsive design, and client-side application logic.

---

## ✨ Features

- 55 Multiple Choice Questions (MCQs)
- Dynamic Question Rendering
- Randomized Answer Options
- Automatic Answer Validation
- Instant Feedback for Each Question
- Final Score Calculation
- Percentage-Based Performance Evaluation
- Highlighting of Unanswered Questions
- Responsive and Modern User Interface
- Smooth Scrolling Navigation
- Refresh Quiz Functionality

## 🚀 How to Run

### Method 1: Open Directly

1. Download or clone the repository.
2. Open the project folder.
3. Double-click `index.html`.
4. The quiz application will launch in your browser.

### Method 2: Using VS Code Live Server

1. Open the project in Visual Studio Code.
2. Install the Live Server extension.
3. Right-click on `index.html`.
4. Select **Open with Live Server**.

## 📊 Assessment Process

1. User selects answers for the quiz questions.
2. The system validates all responses.
3. Correct answers are marked instantly.
4. Incorrect answers display both selected and correct answers.
5. Unanswered questions are highlighted.
6. Final score and percentage are calculated and displayed.
